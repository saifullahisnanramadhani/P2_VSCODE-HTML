Uji coba 3



Memberikan atribut "name" yang sama dan berbeda pada tag input checkbox

- Jika atribut "name" memiliki nilai yang sama, maka hanya satu pilihan dapat dipilih pada satu waktu

- Jika atribut "name" memiliki nilai yang berbeda, maka setiap pilihan dapat dipilih secara independen

- Fungsi type="date" digunakan untuk memungkinkan pengguna memilih tanggal

- Fungsi type="button" dengan atribut (onclick) dapat digunakan untuk menjalankan perintah

Uji coba 8




- Elemen type="url" digunakan untuk memvalidasi dan memastikan bahwa input yang dimasukkan oleh pengguna adalah URL yang valid

- Elemen type="search" digunakan untuk menginput kata kunci pencarian

- Elemen type="tel" digunakan untuk memvalidasi dan membatasi input untuk format nomor telepon tertentu

- Hasil uji coba menunjukkan bahwa elemen type="url" dan type="tel"> berfungsi untuk memvalidasi dan membatasi format input, sementara elemen type="search" memberikan antarmuka pencarian yang dapat digunakan oleh user
